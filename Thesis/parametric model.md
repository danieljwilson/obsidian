# Notes
Parametric methods involve a two-step model-based approach.
1. First, we make an assumption about the functional form, or shape, of $f$
   e.g. a linear model: $$f(X) = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \dots + \beta_p X_p + \epsilon$$
2. After a model has been selected, we need a procedure that uses the training data to fit or train the model.

- Referred to as "parametric" because it reduces the problem of estimating f down to one of estimating a set of parameters.
