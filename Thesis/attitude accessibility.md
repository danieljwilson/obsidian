# Notes
The likelihood that an attitude will be automatically activated from memory on encountering the [attitude object](https://dictionary.apa.org/attitude-object).

- assumed to depend on the strength of the associative link in memory between the representation of the object and the evaluation of the object
- the stronger the memory link between the object and its evaluation, the more quickly the attitude will come to mind
- attitudes that come quickly to mind are believed to be better guides to [[behavior]]

> The latency with which people can respond to an inquiry about their attitudes is considered to be a reflection of the accessibility of the attitude. 
> (Fazio, 1986, p. 215)
