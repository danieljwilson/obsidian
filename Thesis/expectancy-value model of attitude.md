# Notes
States that attitudes are developed and modified based on assessments about beliefs and values.

**Three components**
1. individuals ==respond to novel information== about an item or action by developing a belief about the item or action. If a belief already exists, it can and most likely will be modified by new information ^[seems Bayesian]
2. individuals ==assign a value to each attribute== that a belief is based on ^[not sure what this means? what are the attributes a belief is based on??]
3. an ==expectation is created or modified== based on the result of a calculation based on beliefs and values

- attempts to determine the mental calculations that take place in attitude development
