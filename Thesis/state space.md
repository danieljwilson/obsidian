# Notes
The set of all states reachable from the initial state by any sequence of actions.

**EXAMPLE**
In a 15 puzzle, the state space consists of all the 16!/2 configurations on the board that can be reached from any initial state. The state space can be visualized as a [[causal graph framework|directed graph]] with states, represented as [[node]], and [[action|actions]], represented as arrows between nodes.
