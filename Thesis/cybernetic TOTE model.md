# Notes
Basic structure of strategies.

- T.O.T.E. stands for Test-Operate-Test-Exit.
- The **starting point** is a specific **actual state** and the **target** is a certain **desired state**.
- We compare the actual and the desired state. If this test / comparison shows that the target has not yet been reached, we take this information as feedback and now make certain changes and improvements (operate). Next, we compare the result again with the desired state (test). When the goal is reached, this process is finished (exit). Again, if we find a difference, we operate and test again until the goal is reached.
- model describes the basic structure of human learning in a simple way.
- from Miller, [Galanter](https://en.wikipedia.org/wiki/Eugene_Galanter "Eugene Galanter") and [Pribram](https://en.wikipedia.org/wiki/Karl_H._Pribram "Karl H. Pribram").
