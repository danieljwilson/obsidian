# Notes
Given that, in general, $\hat{f}$ will not be a perfect estimate for $f$ , this inaccuracy will introduce some error. 

- This error is reducible because we can potentially improve the
accuracy of $\hat{f}$  by using the most appropriate statistical learning technique to estimate $f$.
- there is also [[irreducible error]] ($\epsilon$)which cannot be eliminated even if we have a perfect estimate for $f$ (i.e. $\hat{Y} = f(X)$).
