# Notes
The claim that if two events are **correlated**, then either there is a causal connection between the correlated events that is responsible for the correlation or there is a third event, a so called (Reichenbachian) common cause, which brings about the correlation.^[In other words correlation does not imply causation...as there could be a third variable!]
