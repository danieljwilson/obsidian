# Notes

- Set goals often are over-optimistic which can reduce the likelihood that they are achieved
- Setting goals in a cold (e.g., calm or satiated) state can reduce goal attainment because the impact of visceral states (e.g., arousal, hunger) are not taken into account (e.g., Nordgren, van der Pligt, & van Harreveld, 2008).
- Optimistic goal setting can constitute a self-control strategy that helps people to deal with obstacles during goal pursuit.
	- can put more effort into optimistic than realistic goal...better performance even if bigger [[intention-behavior gap]]...
