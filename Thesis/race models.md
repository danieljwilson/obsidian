# Notes
Assumes that a ==go process== begins when the go stimulus occurs and races independently against a ==stop process== that begins when the stop stimulus occurs.

- whichever finishes "first" dictates the response
- provides theoretical framework for understanding the [[stop signal task]]
- assumes context independence (i.e. go process RT should not be affected by presence of stop signal)^[Not certain how to interpret this? Does it mean that if it is a block that does not contain stop signals that the RT should be the same as for a block that does? This seems unlikely. No "caution" necessary. Or does it mean that error "go" responses are the same RT as correct "go" responses?]
- [[race models|independent race model]] also assumes [[stochastic independence]]
