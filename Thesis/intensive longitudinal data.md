# Notes


- intensive longitudinal design: Cross between [[time series]] and [[panel data|panel design]]
- larger number of individuals than time series, but fewer than [[panel data|panel design]]
- [[burst design]] is a hybrid of more traditional [[panel data|panel design]] with intensive measures between panel assessments
- typically want data in long format

##### Analysis
- start with exploratory visualization
- likely occurrence of serially correlated residuals
- potential for transition points and time trends...
	- interruption, or discontinuity in trajectory...deflected by "event"
	- can use piecewise linear models to try and capture this phenomenon
