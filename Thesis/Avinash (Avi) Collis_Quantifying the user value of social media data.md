# Notes
**Looked into how much people value Facebook**
- median value has been falling
- heterogeneity in valuation
	- more time/friends/posting -> higher value
	- women and older people value more
- if you use Instagram/Youtube a lot then you value Facebook less

**Looked into how much users value their data on Facebook**
- Used [[Becker-DeGroot-Marschak mechanism]] (BDM) 
- 5% value at over $10,000
- Median = $750
	- This works out to 73/year and 6.5/month
- Many value at less than $100 (missed the number)
- Demographic differences
	- females less than males (558/1000)
	- black less than white (500/1000)
