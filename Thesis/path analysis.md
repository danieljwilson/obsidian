# Notes
A method to discern and assess the effects of a set of variables acting on a specified outcome via multiple causal pathways.^[Can we really say "causal"?]

- used to describe the directed dependencies among a set of variables
- based on a closed system of nested relationships among variables that are represented statistically by a series of structured linear regression equations
- precursor to and subset of [[structural equations modeling]]
- representation of equations diagrammatically
- developed nearly a century ago by [[Sewall Wright]]
