# Notes
- uses a "margin", which is the shortest distance between the observation and the threshold
- use a [[maximal margin classifier]]
	- sensitive to outliers in the [[training data]] set
- Strategy to deal with outliers = allow misclassifications
	- idea of a **soft** margin
- Has a decision plane and separating planes
- Solution uses [[Lagrange Multipliers]] technique
- High dimensional problem requires hyper-plane
- support vector classifier -> support vector machine
