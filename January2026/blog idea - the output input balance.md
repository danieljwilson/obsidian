### Concept
Idea that in life we are often either in [[output mode]] or [[input mode]].

Writing this post I am in output mode. 
Reading someone else's post I am in input mode.
Making notes about that post puts me back in output mode. (or a combined [[input-output mode]])

### Examples

| Inputs                | Outputs              | Input-Output          |
| --------------------- | -------------------- | --------------------- |
| watching youtube      | making youtube video |                       |
| watching movies/tv    |                      |                       |
| listening to podcasts |                      | having a conversation |
| reading               | writing              |                       |
| buying things         | making things        |                       |
| going out to dinner   | hosting dinner party |                       |
