I was trying to clean my build cache in [[Xcode]] and was given the keyboard shortcut:
`⇧ Shift + ⌘ Cmd + K`

However, this pulled up a Notion quick search pop up...
![[Pasted image 20260127132655.png|500]]

So if you are a Notion user running up against this issue you can just add `⌥ Option` to the front of that sequence:
`⌥ Option + ⇧ Shift + ⌘ Cmd + K`

And you should be good to go!
